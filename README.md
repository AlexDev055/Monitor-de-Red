# Monitor de Red

Este proyecto es una herramienta de monitoreo de red diseñada para analizar el tráfico de red en tiempo real utilizando Python y la librería **Scapy**. El objetivo es capturar paquetes en una red, analizarlos y visualizar datos clave sobre los dispositivos y protocolos activos.

## Características

- Monitoreo de paquetes en tiempo real.
- Identificación de dispositivos y protocolos activos en la red.
- Análisis de paquetes TCP/IP.
- Informes sobre la actividad de la red.

## Tecnologías

- **Python**: Lenguaje de programación principal.
- **Scapy**: Librería de Python utilizada para interactuar con la red y analizar paquetes.



## Requisitos

- Python 3.x
- Librerías necesarias:
  - `scapy`
